Before using Scratch Embed tools on your site, please read over and abide by the following guidelines

Strict Guidleines

Breaking a strict guideline will have you blacklisted from using Scratch Embed in the future

  • Scratch Embed should not be used on innappropriate sites
    Innappropriate sites include any sites which violate the Scratch Community Guidelines.
  • Do not abuse the service
    This includes, but isn't limited to: using Scratch Embed on extremely high traffic sites, using large amounts of embeds on a single page, attempting to find exploits in the site, embedding on a page which is constantly refreshing, etc.

Flexible Guidelines

Flexible guidleines should be abided by whenever possible, but there is room for a bit of flexibility

  • Try to limit Scratch Embed to 7 embeds per page
    This ensures that it isn't overwhelmed with requests by a single site

If these guidelines aren't followed by the majority of users, then a whitelist will have to be implemented, where users will have to register sites where Scratch Embed will be allowed to be used.
