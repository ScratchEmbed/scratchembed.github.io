Documentation

Scratch Embed is 100% frontend, and extremely simple to use. Each tool has a homepage which will provide you with an embed code. To use the code, just copy and paste it into your website's code. This will create an embed which displays the aspect of Scratch that you have chosen to display.

For more detailed and specific instructions, visit the page for the tool that you want to use, and click on the "Instructions" button.
